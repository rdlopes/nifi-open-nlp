PreTrainedLanguageDetector

  • Evaluation
  • Output

This processor wraps the evaluation from OpenNLP Language Detector. Language Detector parses text to yield the closest language trained matching the content. It loads its model from a file path.

Evaluation

PreTrainedLanguageDetector runs against full text, meaning it will read the flow file content.

Output

PreTrainedLanguageDetector will enrich flow file attributes with

  • nlp.language.detector.languages.list: the languages list
  • nlp.language.detector.languages.best: the language with highest probability
  • nlp.language.detector.supported.list: the list of supported languages
