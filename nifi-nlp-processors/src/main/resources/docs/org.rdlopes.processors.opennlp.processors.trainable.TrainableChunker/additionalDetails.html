OpenNLP Chunker

This processor wraps the evaluation from OpenNLP Chunker trained with configured data.

It splits text into syntactically correlated parts for sentence analysis.

Configuration Properties

As a trainable processor, the NLP model will be built at validation time. As such, it requires training data to generate the model. The training data can be provided as a file or directly as a string.
